Tagging
--------
##### Annotated Tag
```
$ git tag -a v1.0 -m "Version 1.0"
```

##### Tagging a specific commit
```
$ git tag -a v1.0 <hash>
```

Pushing
--------
##### Different local branch name
```
$ git push upstream local-branch:remote-branch
```

Branches
--------
##### Comparing
```
$ git log branch1..branch2
$ git diff branch1..branch2
```
